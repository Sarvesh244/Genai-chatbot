## 📌 Project Overview
**CareerGenie** is a domain-specific, production-ready Generative AI chatbot built to provide intelligent career advice, resume reviews, and interview preparation. 

Unlike basic API wrappers, this project is engineered with real-world AI development standards, featuring a clean modular architecture, sliding-window conversational memory, secure environment configurations, runtime logging, and a permanent background deployment on AWS EC2.

🌐 **Live Demo:** (https://genai-chatbot-by-sarvesh-mote.streamlit.app/)  

---

## ✨ Key Features
* **Advanced Prompt Engineering:** Utilizes structured, role-based system prompts with strict domain constraints to ensure the bot stays in character as an expert career advisor.
* **Contextual Memory Management:** Implements a multi-turn conversation tracker with sliding-window optimization to maintain context without exceeding token limits.
* **Clean Architecture:** Strict separation of concerns (UI, API services, memory management, and prompt configuration).
* **Enterprise-Grade Security:** API keys and environment variables are strictly managed outside the codebase using `.env`.
* **Robust Logging & Monitoring:** Tracks API latency, token usage simulations, and runtime errors to `app.log`.
* **Cloud Deployment:** Hosted on an AWS EC2 instance (Ubuntu) running as a permanent background process.
